# Sistema Inteligente de Rotas de Petrolina

Projeto acadêmico desenvolvido para a disciplina de Estrutura de Dados.

## Tema

Grafos em Estrutura de Dados.

## Descrição

Este projeto representa pontos importantes da cidade de Petrolina usando grafos.

Cada local é representado por um vértice.  
Cada ligação entre dois locais é representada por uma aresta.  
Cada aresta possui um peso, que representa o tempo médio de deslocamento em minutos.

O sistema permite escolher uma origem e um destino, e calcula a melhor rota usando o algoritmo de Dijkstra.

## Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS
- JavaScript
- Cytoscape.js

## Algoritmo utilizado

O algoritmo utilizado foi o Dijkstra.

Ele calcula o menor caminho entre dois vértices em um grafo ponderado.

## Funcionalidades

- Interface web
- Cálculo da menor rota
- Visualização do grafo
- Destaque do melhor caminho
- Exibição do tempo estimado
- Exibição da quantidade de vértices e arestas
- Passo a passo do algoritmo de Dijkstra
